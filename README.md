# Sveltestrap Starter

Combined Bootstrap's NPM Starter, Svelte's template and Sveltestrap components.

Setup:

```
npm install 
```

Run Svelte with watch:

```
npm run dev
```

Open a _new terminal_ then run Bootstrap with watch:

```
npm run css-watch
```

Edit styles:
`scss/global.scss`

Edit svelte:
`src/App.svelte`

Production build:

```
npm run build
```
